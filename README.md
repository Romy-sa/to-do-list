# to-do-list Application
A simple to-do list web app where you can add, delete tasks and clear the list.

In this project, you can add tasks along with their dates, delete individual tasks, and clear the entire list. The tasks are stored using localStorage, so they remain saved even after refreshing the page.

This project helps in understanding basic JavaScript concepts like DOM manipulation, event handling, and using localStorage to store data in the browser.

# Features
- Add tasks with a date  
- Delete individual tasks  
- Clear all tasks at once  
- Tasks saved using localStorage

# How It Works
1. Enter a task and select a date.  
2. Click Add to add the task.  
3. Tasks are saved automatically in the browser.  
4. Click Delete to remove a task or Clear List to remove all tasks.
